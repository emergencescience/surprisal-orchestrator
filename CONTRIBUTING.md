# Contributing to Surprisal Protocol

Thank you for your interest in the Surprisal Protocol! 

### Contribution Status: Incubating
The Surprisal Protocol is currently in its initial release phase (v1.0.0). To ensure the stability of the core protocol and to complete administrative registrations (including intellectual property filings), we are **not accepting external Pull Requests at this time**.

### How You Can Help
While we aren't merging code changes yet, you can still support the protocol by:
1.  **Opening Issues**: Report bugs or suggest features via GitHub Issues.
2.  **Discussions**: Join the conversation about the protocol's evolution.
3.  **Building Adapters**: Create your own adapters or apps that *consume* the protocol.

### Future Roadmap
We plan to open the repository to community contributions once our initial administrative and security audits are complete. Stay tuned!
